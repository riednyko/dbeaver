DBeaver Roadmap 2012

Open-source

Choose open-source license.
Make public access (read-only) to version control and bug-tracking system.
Make a team and grant commit rights to members.

Client-side scripting

Add support of client-side scripting: velocity style macros-like commands which are executed on client side and allows additional data handling across multiple databases.

Script management

Improve current script management system

Database plugins

Add new extensions to support native database features for:
  • IBM DB2
  • MS SQL Server
  • PostgreSQL

Help

  1. Replace eclipse help with own lightweight help system
  2. Context help for dialogs/editors

Eclipse plugin mode

Support Eclipse plugin mode.

Localization

  1. Italian, German, Russian, Chinese. Actual language set depends on requests and available team members.

Metadata editor

  1. Improve metadata editors (stored procedures, administrative tools, etc).

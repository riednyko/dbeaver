Database URL Notes
Daffodil DB / One$DB http://sourceforge.net/projects/daffodildb/ Daffodil/One$DB JDBC driver included in database archive
DB2 http://www.ibm.com/software/data/db2/express/download.html IBM DB2 driver can be downloaded only as part of database server distribution
Derby / JavaDB http://db.apache.org/ JDBC driver is a part of Derby database
Firebird http://www.firebirdsql.org/index.php?op=files&id=jaybird Jaybird - Firebird/Interbase Class 4 JCA-JDBC Driver
H2 http://www.h2database.com/ JDBC driver is a part of H2 database
HSQLDB http://hsqldb.org/ JDBC driver is a part of HyperSQL database
HXTT Drivers http://www.hxtt.com/products.html HXTT drivers for MS Access, Excel, CSV, DBF, Paradox, Cobol
Informix http://ibm.com/software/data/informix/downloads.html IBM Informix JDBC driver
Ingres http://www.ingres.com/downloads/ingres-database#jdbc Standard Ingres JDBC driver
Interbase http://info.borland.com/devsupport/interbase/interclient/ Borland InterClient driver
Mckoi http://developer.mimer.com/ JDBC driver is a part of Mckoi database
Mimer SQL http://developer.mimer.com/ Mimer standard JDBC driver
MySQL http://www.mysql.com/downloads/connector/j/ MySQL Connector/J
MS SQL Server http://jtds.sourceforge.net/ JTDS SQL Server/Sybase driver
http://msdn.microsoft.com/en-us/sqlserver/ff658549 Microsoft JDBC driver
OpenBase http://www.openbase.com/index.php/products/downloads OpenBase JDBC driver
Oracle http://www.oracle.com/technetwork/database/features/jdbc/ Thin or OCI driver. OCI driver requires Oracle client installation
PostgreSQL http://jdbc.postgresql.org/ Standard PostgreSQL JDBC driver
SAP DB http://www.sapdb.org/sap_db_jdbc.htm SAP DB/ SAP MAXDB drivers
SmallSQL http://www.smallsql.de/download.html JDBC driver is a part of SmallSQL Database
SQLite http://www.sqlite.org/cvstrac/wiki?p=SqliteWrappers Set of available SQLite wrappers including pure Java JDBC implementations
Sybase http://jtds.sourceforge.net/ JTDS SQL Server/Sybase driver
http://www.sybase.com/products/allproductsa-z/softwaredeveloperkit/jconnect Sybase jConnect driver

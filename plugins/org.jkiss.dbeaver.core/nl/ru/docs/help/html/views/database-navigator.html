Окно базы данных

Окно базы данных позволяет Вам просматривать все соединения с базами данных проекта. Когда Вы раскрываете дерево объектов соединения, DBeaver автоматически соединяется с удалённой базой данных. Структура объектов в дереве соединения зависит от драйвера базы данных. Как правило, соединения содержат каталоги, каталоги содержат схемы, схемы - таблицы, индексы, процедуры и т.д. Элементы дерева (базы данных) могут быть открыты, отредактированы, удалены и т.д. Возможные дополнительные команды доступны в контекстном меню или на панели инструментов.
